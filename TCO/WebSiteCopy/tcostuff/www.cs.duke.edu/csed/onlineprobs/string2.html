Weekly Problem string 2

Problem Statement

A string is prefix-suffix-swapped (or ps-swapped) if some suffix is moved so that it becomes a prefix. For example, the string "flapjack" can be ps-swapped on the suffix "jack" to become "jackflap", on the suffix "ack" to become "ackflapj" and on other suffixes as well.

Given a string s, and an integer index index, return the ps-swapped string created by moving the suffix that starts with the index-th character so that it is a prefix.

Definition

  • Class: none used in this problem
  • Method: prefixSuffix
  • Parameters: string int
  • Returns: string
  • Method signature (see below)

Constraints

  • the value of index will be between 0 and one less than the length of the string, i.e., it will be the index of a valid suffix of the string parameter
