Weekly Problem date2

Problem Statement

January 1 is the first day of any year. February 1 is always the 32nd day of any year. March 1 is either the 60th or 61st day of a year depending on whether the year is a leap year.

Given two integer values: a number between 1 and 365 (or 366 for a leap year) representing a specific date, and a year, your function will return a string "Sunday", "Monday", ... or "Saturday" (quotes for clarity) that is the day of the week on which the specified date occurs.

For example, given 1 2003 your method should return "Wednesday" since January 1 of 2003 is a Wednesday.

Definition

  • Class: none used in this problem
  • Method: dayName
  • Parameters: int, int
  • Returns: string
  • Method signature (see below)

Prototype

(be sure you have the necessary include files) #include <string> using namespace std; #include "date.h" string dayName(int days, int year) { // fill in code here }

Constraints

  • days is between 1 and 366.

  • days will be 366 only if year represents a leap year.

  • year will be between 1800 and 2500.
