Weekly Problem string1

Problem Statement

The square length of a string is the square of the number of characters in the string. The square length of "hello" is 25, the square length of "a" is 1, and the square length of "apple sauce" is 121.

Given a string, write a function that returns the string's square length.

Definition

  • Class: none used in this problem
  • Method: squareLength
  • Parameters: string
  • Returns: int
  • Method signature (see below)

Prototype

(be sure you have the necessary include files) #include <string> using namespace std; int squareLength(string str) { // fill in code here }

Constraints

  • The square length of the parameter s will be less than INT_MAX, you do not need to worry about integer overflow.
