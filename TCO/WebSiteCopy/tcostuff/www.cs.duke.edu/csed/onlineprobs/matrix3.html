Matrix 3, Most High Games

Problem Statement

Your task is to return the name of the player with the most high game scores. A player has the high game score if he/she scored more points than anyone else in that game. There can be several players with the high game score in one game. If there is more than one player with the most high scores, then you should return a string with the names of all the players with the most high scores, with names separated by a "-". The order of these names should be in the same order they appear in the array they are given in. A player's name should only appear once.

You are given a vector called names of player names and a matrix called scores with scores for some sport, where each row in the matrix represents the scores of a player on the team and each column is all the players's scores in a game. Thus, the entry scores[k][j] in the matrix is the number of points scored by player k for game j. The corresponding player's name is in names[k].

For example, suppose the game is Duke basketball, there are 14 players on the team, and they have played three games so far. The first column below is the Names vector and the next three columns are in the matrix scores. 

Deng         18   12    17
Horvath      17    0    17
Redick        9   16    23
Ewing        13   23    6
Melchionni   21    9    3
Dockery       9    6    4
Duhon        12   11    6
Williams     16   12   17
Borman        0    0    0
Davidson      0    0    0
Randolph     23   12   18
Thompson     15    4    0
Johnson       0    0    2
Means         0    0    0

The high scorer for the first game is Randolph with 23 points. The high scorer for the second game is Ewing with 23 The high scorer for the third game is Redick with 23. The string returned should be "Redick-Ewing-Randolph" as each of these players got the highest score in some game and no player got the highest score in a game more than once.
