Matrix 1, Highest Scoring Game

Problem Statement

Your task is to return the total score of the highest game. You are given a matrix called scores with scores for some sport, where each row in the matrix represents the scores of a player on the team over several games and each column is the individual scores in a game. Thus, the entry scores[k][j] in the matrix is the number of points scored by player k for game j. The total score of a game is the sum of the individual scores in the game.

For example, suppose the game is Duke basketball, there are 14 players on the team, and they have played three games so far. The three columns are in the matrix scores. 

     18   12    17
     17    0    17
      9   16    23
     13   23    6
     21    9    3
      9    6    4
     12   11    6
     16   12   17
      0    0    0
      0    0    0
     23   12   18
     15    4    0
      0    0    2
      0    0    0

The max score returned should be 153 , the sum of the first column, the game with the most points.

Definition

  • Class: none used in this problem
  • Method: highGameScore
  • Parameters: tmatrix<int>
  • Returns: int
  • Method signature (see below)

Prototype

(be sure you have the necessary include files) #include "tmatrix.h" int highGameScore(tmatrix <int> & scores) { }
