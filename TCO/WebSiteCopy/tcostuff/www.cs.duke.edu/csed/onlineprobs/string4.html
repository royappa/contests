Weekly Problem string4

Problem Statement

Given a URL for a web page such as www.cs.duke.edu/help.html, where the URL ends with a page whose name has a .html suffix, return a string that contains the type of the URL domain, e.g., com, edu, org, net; and contains the root-name (without the .html suffix) of the web page. The domain and root-name should be separated by a colon, e.g., "edu:help".

The domain name is whatever comes after the last period of the prefix that occurs before the first slash /; and the root-name is whatever comes before the suffix .html and after the last slash.

Hint: The string member function rfind will be useful, see Tapestry for details.

Definition

  • Class: none used in this problem
  • Function/Method: parseUrl
  • Parameters: string
  • Returns: string
  • Method signature (see below)

Prototype

(be sure you have the necessary include files as shown) #include <string> using namespace std; string parseUrl(string url) { // fill in code here }

Constraints

  • Parameter url contains at least one slash / character.

  • Parameter url contains at least one period before the first slash.

  • Paramter url has a valid web-page name with a .html suffix after the last / in url.

Examples

Returns:  "edu:tests"

  4.

Returns: "gov:cps"
