Weekly Problem date1

Problem Statement

January 1 is the first day of any year. February 1 is always the 32nd day of any year. March 1 is either the 60th or 61st day of a year depending on whether the year is a leap year.

Given a date specified in American standard month, day, year format (using integer values), your function will return what day of the year the date represents.

For example, given 1 1 2003 your method should return 1.

Definition

  • Class: none used in this problem
  • Method: dayNumber
  • Parameters: int, int, int
  • Returns: int
  • Method signature (see below)

Prototype

(be sure you have the necessary include files) #include "date.h" int dayNumber(int month, int day, int year) { // fill in code here }

Constraints

  • month, day, and year parameters will represent a real date, e.g., parameters 2 29 2001 are not permitted since 2001 is not a leap year.

Examples

  1.   1  1  2003

Returns:  1
    From above 

  2.     

  3  1  2000

Returns: 61

    The year 2000 is a leap year. 

  3.    
  3  1  1900

Returns:60

    The year 1900 is not a leap year. 

  4. 
  12 31  1999

Returns: 365
