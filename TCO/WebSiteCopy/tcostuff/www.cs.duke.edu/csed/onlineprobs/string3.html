Weekly Problem string3

Problem Statement

Given a time in one of the following three string formats return the number of seconds represented by the time as an int. The quotes below aren't part of the strings, but are shown for clarity.

"00" hours
"00:00" hours and minutes
"00:00:00" hours, minutes and seconds

Hint: The function atoi from the include file "strutils.h" will be useful in solving this problem. The atoi function converts a string like "23" to its numeric, int equivalent, i.e., 23. See page 289 in Tapestry for details.

Definition

  • Class: none used in this problem
  • Method: seconds
  • Parameters: string
  • Returns: int
  • Method signature (see below)

Prototype

(be sure you have the necessary include files as shown) #include <string> using namespace std; #include "strutils.h" int seconds(string str) { // fill in code here }

Constraints

  • The paramete str will be in one of the three formats shown.
